> TypeScript definitions for [Defold](https://defold.com/)

## Installation

1. Get this package from npm

```bash
yarn add @ts-defold/types --dev
# or
npm install @ts-defold/types --save-dev 
```

2. Modify your `tsconfig.json`

```diff
{
  "compilerOptions": {
+    "types": ["@ts-defold/types"]
  }
}
```
